Notes:
Thank you for trying out Gnome Tower! The current version is likely buggy, and not very user friendly. If you have problems, find bugs, or would just like to check out the latest happenings, you can check out our Discord channel.

To play solo (PCVR):
- Run gnomes.exe
- Hit "host local world" (if it doesn't automatically)
- It will begin a session using the included "save0" as a save
- If no "save0" present, it will create one

To host a server (PCVR):
- Enable port forwarding on your router
- Forward port 54000 (default) to your PC
- Follow "play solo" instructions to play alongside your server, OR instead run gnomes_pcserver.exe for standalone server

To join a server (PCVR):
- Run gnomes.exe
- Hit "Join world"
- Type in server IP address and port, separated by a colon (something like 68.112.183.12:54000)
- Hit the right arrow to attempt a connection

To join a server (Quest- cannot act as host!):
- "Side load" gnometower-release.apk onto your quest
- follow "join a server PCVR" instructions


HOW TO PLAY:
- Movement:
- - Forward/Backward on either joystick will hop your character forward or back
- - Left/Right will snap turn
- - Grip anywhere to pull yourself around
- - Grip with two hands to 1. set pivot, and 2. rotate around that pivot (MUST BE ENABLED IN SETTINGS)
- Placement:
- - Selection (Default):
- - - Use trigger on gnomes, drag and release in another location, to command them to path (for fun/debugging)
- - Hand:
- - - Use trigger to grab mobs
- - ? Gnome:
- - - Tell gnomes to reconsider current jobs (necessary when something fails to be built)
- - box Gnome:
- - - Set volume to instruct gnomes to build [b toggles set/clear command]
- - explode Gnome:
- - - Set volume to instruct gnomes to destroy [b toggles set/clear command]
- - Volume:
- - - Hold trigger on one controller to preview a single voxel placement; release to place
- - - Hold trigger on both controllers to preview a selection range; release to place entire range
- - - B cancels selection preview
- - Paint:
- - - While holding trigger on either controller, it will continuously place voxels at that position
- - Mask:
- - - The first option ("add") will place blocks only where no block exists
- - - The first option ("set") will place blocks wherever specified (overwrites existing blocks)
- - - The third option ("modify") will overwrite blocks only where a block already exists
- - Size:
- - - paintbrush size
- - B toggles between place and delete
- Menu:
- - Hold A to show menu
- - Use other hand's beam to select options


Download Contents:
gnomes.exe Combined client/server for standalone PCVR (Windows SteamVR or Oculus runtime). Other clients can join this just like standalone server.
[ USE THIS TO CLICK AND PLAY ]
gnomes-server.exe Standalone Server (Windows PC only). Default port is 54000. Can run from command line as `path\to\gnomes-server.exe 1234` to (eg) set the port to 1234
gnometower-release.apk Quest version.
[ REQUIRES RUNNING gnomes.exe or gnomes-server.exe INSTANCE TO CONNECT TO ]
resources Game assets. Run PCVR at the same folder heirarchy as this folder (should be fine to run from unzipped structure).
save0 World save file. Server saves world on close window.
settings.txt Game settings. Set "forceFrameSync 1" if using the oculus PCVR runtime.
